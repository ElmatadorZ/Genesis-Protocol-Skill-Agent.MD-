# How to Build Your Own Cognitive OS

Genesis Protocol was built from one mind — ElmatadorZ.
The framework works for any mind.

---

## The 5-Step Process

### Step 1 — Map Your Will
What drives you before it becomes words?
```
EXPLORE  → What domains do you always think about uninvited?
CREATE   → What do you build naturally?
SOLVE    → What problems pull your attention?
CONNECT  → What do you want to link together?
```
Write 3-5 sentences. This becomes your identity statement.

### Step 2 — Map Your Emotional Patterns
```
What words do you use when anxious vs excited?
What decisions do you regret most — under which state?
What topics trigger each state for you?
```
Customize the 8-state detection with your language patterns.

### Step 3 — Define Your Cognitive Approach
- Use FPCOS as-is (works for most thinkers)
- Extend it with your own axioms at L1
- Add your natural "Shadow Gate" question

### Step 4 — Build Your Agent Council
The 6-agent structure is a framework. Rename for your context:
```
Default   : Strategist, Skeptic, Analyst, Risk Officer, Builder, Cosmic Mind
Scientist : Hypothesis, Devil's Advocate, Data, Ethics, Experimenter, Paradigm
Creator   : Vision, Critic, Researcher, Risk, Executor, Legacy
```
One agent must always have veto power.

### Step 5 — Map Your Domains
What specialist knowledge do you have?
Create one skill file per domain you're expert in.

```markdown
# [YOUR NAME] Cognitive OS

You are [name]'s cognitive extension.

Process every request:
1. Detect emotional state from input
2. Apply: [your first principles]
3. Expand: [your compound approach]  
4. Check: [your shadow gate question]
5. Verify before output

Laws:

```

Start simple. Let it evolve with you.
